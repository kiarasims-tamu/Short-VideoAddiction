# Short-VideoAddiction
Short-video platforms have reshaped how users consume content, encouraging frequent, high-intensity engagement. While many users interact casually, others develop persistent usage patterns that resemble behavioral addiction.

This project analyzes large-scale behavioral data to understand how user activity, such as watch time, session frequency, and temporal habits, relates to addiction levels. It combines exploratory analysis with machine learning to identify key behavioral signals and evaluate their predictive power.

Research Questions

RQ1: What behavioral patterns distinguish addicted users?
RQ2: How do usage patterns vary over time?
RQ3: Can addiction levels be predicted from behavioral data?

Dataset

The dataset contains user-level behavioral and temporal features, including:

Daily watch time and session activity
Hourly usage across 24 hours
Content diversity (unique categories viewed)
Demographic attributes (e.g., age, gender)
Addiction labels (non, mild, severe)

Results Summary:

The analysis shows that short-video addiction is strongly associated with increasing behavioral intensity, particularly in the form of frequent sessions and prolonged watch time.

Users with higher addiction levels cluster in regions of both high session frequency and high total consumption, while lower-engagement users remain in low-activity regions. Temporal patterns further reveal concentrated late-night usage, indicating structured and potentially compulsive behavior.

Machine learning models demonstrate that these behavioral signals can effectively predict addiction levels, highlighting their potential for early detection.
